# Settings

The Settings tab groups everything you can tune, section by section.

## Appearance

- **Color mode** — follow the **system**, or force **light** or **dark**.
- **Language** — choose from English, German, Spanish, French, Italian, Japanese, Korean, or Chinese, or keep the **system default**.

## Browsing

- **Open in Custom Tabs** — open links in the system browser's Custom Tabs instead of the in-app WebView.

## Webpage fetching

- **Custom User-Agent** — set the User-Agent string Magic Feed uses when fetching page metadata. Useful for sites that serve different markup to different browsers. You can edit it freely or reset to the default at any time.

## Shared text

Control what's included when you share a link as text:

- **Include title** — add the page title before the URL.
- **Include description** — add the page description before the URL.

## Downloads

- **Download location** — choose the folder where images are saved, or reset to the default Pictures folder.

## Data management

- **Backup location** — choose a folder for backups so they survive uninstall, or reset to app storage. See [Backup & Restore](./backup-restore).
- **Auto backup to local storage** — keep a backup current automatically.
- **Auto backup to Google Drive** — cloud backups without lifting a finger.
- **Clear all feeds** — remove every saved feed from your device (with confirmation).

## Support

- **How Magic Feed works** — an in-app explainer.
- **Report an issue** — jumps to GitHub issues.
- **App version** and a link to the **GitHub repository**.
